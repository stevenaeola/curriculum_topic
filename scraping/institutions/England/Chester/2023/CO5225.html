Scheduled hours Placement Hours Independent Guided study
40 0 160

Students are expected to attend all lectures, tutorials and workshops.

  • Abstraction, Encapsulation and Object Composition
  • Access modifiers
  • Polymorphism, dynamic dispatch
  • Interfaces and Abstract classes
  • Object composition
  • Single Responsibility principle, Liskov Substitution Principle
  • Type systems (static /dynamic typing)
  • Type conversion & casting
  • Use of libraries classes / packages and APIs
  • Exception raising, propagation and handling
  • Further refactoring techniques
  • Integration & System testing
  • Version control

  • To introduce some of the more esoteric concepts of Object Oriented Programming
  • To develop techniques for avoiding, handling and testing for errors
  • To introduce type systems and static & dynamically typed languages

  • Tutor directed and self-directed study.
  • Lectures, workshops/tutorials.
  • Online learning as directed. Demonstrations and practical exercises.
  • Formative feedback will be given during contact time, by email and via discussion boards where appropriate.

  1. Translate a programme specification into a working piece of software
  2. Apply more advanced principles of OOP in the development of a piece of software
  3. Develop software which integrates third party code
  4. Refactor code into an alternate or improved form
  5. Identify key differences between dynamically and statically typed languages
  6. Discuss, in depth, key principles of Object Oriented Programming beyond that of basic class design
